# 💧 HydroTrack - Drink Water Reminder

HydroTrack is a modern Android application built with Kotlin that helps users maintain healthy hydration habits by tracking daily water intake and sending timely reminders to drink water.

## 📱 Features

### 🚰 Water Intake Tracking
- Add water consumption with a single tap
- Track daily water intake progress
- View total glasses consumed throughout the day
- Simple and intuitive user interface

### 🔔 Smart Reminders
- Custom reminder intervals
- Notification alerts to drink water
- Background reminder scheduling
- Easy reminder management

### 📊 Daily Progress Monitoring
- Track completed water glasses
- Visual hydration progress
- Daily goal management
- Motivation to maintain consistency

### 🎨 Modern UI
- Clean Material Design interface
- User-friendly experience
- Lightweight and fast performance
- Responsive layouts

## 🛠️ Tech Stack

- **Language:** Kotlin
- **Platform:** Android
- **Architecture:** MVVM
- **UI:** Jetpack Compose / XML Layouts
- **Notifications:** Android Notification Manager
- **Storage:** SharedPreferences / Room Database
- **Background Tasks:** WorkManager
